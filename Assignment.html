Types of Html tags

There are many types of htlm tags which are namely...

1- html>: Represents the root element of an HTML document

2- base>: Specifies the base URL for relative URLs

3- head>: Contains machine-readable information about the document

4- link>: Specifies relationships between the document and external resources

5- meta>: Represents metadata that can't be represented by other HTML meta-related elements

6- style: Contains style information for a document or part of a document

7- title: Defines the document's title

8- body: Represents the content of an HTML document

9- address: Indicates contact information

10- article: Represents a self-contained composition

11- aside: Represents indirectly related content

12- footer: Represents a footer

13- header: Represents introductory content

14- h1-6: Represent section headings

15- hgroup>: Represents a heading grouped with secondary content

16- main>: Represents the dominant content

17- nav>: Represents navigation links

18- section>: Represents a generic standalone section

19- blockquote>: Indicates an extended quotation

20- dd>: Provides a description or value

21- div>: Generic container for flow content

22- dl>: Represents a description list dt>: Specifies a term

23- figcaption>: Represents a caption figure>: Represents self-contained content

24- hr>: Represents a thematic break li>: Represents an item in a list

25- menu>: Represents an unordered list ol>: Represents an ordered list p>: Represents a paragraph pre>: Represents preformatted text ul>: Represents an unordered list

26- a>: Creates a hyperlink abbr>: Represents an abbreviation b>: Draws attention to content bdi>: Isolates text from surrounding text bdo>: Overrides text direction br>: Produces a line break

27- cite>: Marks up the title of a cited work code>: Displays computer code data>: Links content with machine-readable translation

28- dfn>: Indicates a term being defined

29- em>: Marks text with stress emphasis

30- i>: Represents idiomatic text

31- kbd>: Represents user input mark>: Represents highlighted text q>: Indicates a short inline quotation rp>: Provides fall-back parentheses rt>: Specifies ruby text ruby>: Represents ruby annotations s>: Renders text with strikethrough samp>: Represents sample output small>: Represents side-comments span>: Generic inline container

32- strong>: Indicates strong importance sub>: Specifies subscript text
